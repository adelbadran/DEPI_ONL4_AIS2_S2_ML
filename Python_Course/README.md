## üìö Course Curriculum

### **Session 1: The Building Blocks**
* **Output & Styling:** `print()` functions, comments, and escape sequences.
* **Strings:** Formatting methods (f-strings) and string manipulation.
* **Data Types:** Primitive types and memory address identification.
* **Intro to Collections:** Basics of Lists, Tuples, and Sets.

### **Session 2: Data Structures & Logic**
* **Collections Deep Dive:** Advanced methods for Lists, Tuples, and Sets.
* **Dictionaries:** Mapping data using key-value pairs.
* **Input/Output:** Capturing user input and performing operations.
* **Control Flow:** Conditional logic (`if-else`) and iteration (Loops).

### **Session 3: Modular Programming**
* **Functions:** Creating reusable code blocks and parameter validation.
* **Libraries:** Using the `random` module for data generation.
* **File I/O:** Initial techniques for reading and writing local files.

### **Session 4: System Architecture**
* **Advanced File Handling:** Context managers and data persistence.
* **OOP (Object-Oriented Programming):**
    * Classes and Objects
    * Attributes and Methods
    * Inheritance & Encapsulation
